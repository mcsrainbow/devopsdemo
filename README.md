Instruction
---
1. Clone the repo<br>
`cd ~`<br>
`git clone https://github.com/mcsrainbow/devopsdemo`<br>

2. Configure the environment<br>
`cd devopsdemo/scripts`<br>
`./requirements.sh all`<br>

3. Run the app<br>
`cd ~/devopsdemo`<br>
`. flask/bin/activate`<br>
`./run.py`<br>
